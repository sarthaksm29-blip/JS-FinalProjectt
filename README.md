# HealthMate – Preventive Health Tracker

HealthMate is a responsive web-based preventive health tracking application developed using **HTML5, CSS3, and JavaScript**.  
The system allows users to track important health metrics such as **BMI, blood pressure, and sugar levels**, store records, and view a health summary dashboard.

This project was developed as part of the **HTML5, CSS3, JavaScript Case Study – Semester II** for **B.Tech CSE (2025–29)** at **ITM Skills University**.

## Project Objective

The objective of this project is to create a simple preventive health monitoring platform where users can:

- Track BMI automatically
- Record daily health metrics
- View health summary reports
- Simulate doctor consultation booking
- Explore subscription plans for premium features

The system demonstrates the practical use of **front-end web technologies** to build a real-world healthcare application.

---

## Technologies Used

- **HTML5** – Page structure and forms
- **CSS3** – Styling, layout, responsive design (Grid & Flexbox)
- **JavaScript (ES6)** – Logic, BMI calculation, DOM manipulation
- **LocalStorage API** – Persistent health record storage

---

## Project Features

### 1. Home Page
- Introduction to HealthMate
- Feature cards highlighting system capabilities
- Start Tracking call-to-action

### 2. Health Tracker
Users can input:

- Date
- Height
- Weight
- Blood Pressure (Systolic / Diastolic)
- Sugar Level

Features:
- Automatic **BMI calculation**
- Health records saved to **LocalStorage**
- Dynamic records table

---

### 3. Health Reports
Dashboard displaying:

- Average BMI
- Blood Pressure
- Sugar Level
- Overall Health Status

Provides a quick summary of user health metrics.

---

### 4. Doctor Consultation
Displays specialist profiles:

- Dr. Mehta – Cardiologist
- Dr. Sharma – General Physician
- Dr. Patel – Nutritionist

Users can simulate booking appointments with doctors.
